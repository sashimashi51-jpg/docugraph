# Docugraph

Docugraph е просто и мощно приложение за работа с вашите документи и знания. Този проект е специално оптимизиран за Windows.

## Инсталация и стартиране (Windows)

За да инсталирате и стартирате Docugraph, просто изпълнете следните стъпки:

1. Отворете папката `scripts` в проекта.
2. Стартирайте файла `run_windows.bat` (може да кликнете два пъти върху него или да го стартирате през командния ред).
3. Изчакайте инсталацията на нужните компоненти да приключи.
4. Приложението ще се стартира автоматично и ще се отвори във вашия браузър.

## Обновяване

Ако искате да обновите Docugraph до най-новата версия:

1. Отворете папката `scripts`.
2. Стартирайте файла `update_windows.bat`.

## Бележка

Този проект е оптимизиран и изчистен от всички Linux и macOS специфични файлове. Той е предвиден за работа единствено на Windows операционни системи. Проектът вече носи името **Docugraph**!
